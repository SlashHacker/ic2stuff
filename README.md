Compatibility
=============
This version of ic2stuff is for Forge 9.11.0 / MC 1.6.4 and IC2 version 2 (experimental).




Installing
=============

Just copy the zip file into the .minecraft\mods folder. Typically located at %AppData%\.minecraft\mods in Windows.
